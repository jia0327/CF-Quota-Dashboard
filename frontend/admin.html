参数获取说明

Account Name：自定义标签，任意命名，用于在仪表盘中区分多个账号。

Account ID：登录 Cloudflare Dashboard → 进入任意站点「概览 / Overview」→ 右侧边栏查看 Account ID（32 位字符）；或打开 Account Home，从浏览器地址栏 URL 中获取（/accounts/…）。

API Token (read-only)：Dashboard → My Profile（我的个人资料） → API Tokens → Create Token → Create Custom Token，为被监控账号创建只读 Token。

最低必需权限（缺一不可）： Account → Account Analytics → Read（Workers、D1 读/写/存储、KV、R2、Workflows 等 GraphQL 用量）
Account → D1 → Read（REST 查询 D1 数据库个数）

建议一并勾选（按需）： Cloudflare Pages: Read · Workers Scripts: Read · Workers KV Storage: Read · Workers R2 Storage: Read · Queues: Read · Hyperdrive: Read · Vectorize: Read · Account Settings: Read（Verify 时显示账号名）

保存前建议先点击「验证凭据」，确认 Token 有效且权限足够。
